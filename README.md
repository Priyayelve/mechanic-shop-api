# Mechanic Shop API

This is a Flask REST API for managing customers, mechanics, service tickets, and inventory.

## Features
- Customer routes
- Mechanic routes
- Service ticket routes
- Inventory blueprint
- JWT token authentication
- Protected route using Bearer Token
- Rate limiting with Flask-Limiter
- API testing with Postman

## How to Run

Install dependencies:

```bash
pip3 install flask flask-sqlalchemy flask-marshmallow marshmallow-sqlalchemy flask-limiter python-jose
